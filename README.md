# Project setup
### Clone
```
git clone https://github.com/equationslayer12/frequency_analysis_battle.git
npm install
cd backend
pip install -r requirements.txt
```
### Run backend
inside `/backend/`:
```
uvicorn main:app --reload
```
### Run front-end
```
npm run serve
```
